Isis automatically fixes issues found by Ma'at — lint errors, vet warnings, formatting violations, and coverage gaps. She is the autonomous remediation engine that heals your codebase so you can focus on building, not fixing.

Features

🔧Auto-Fix Lint

Automatically resolves golangci-lint findings — unused variables, missing error checks, inefficient patterns, and style violations

🩺Auto-Fix Vet

Resolves go vet warnings including shadowed variables, unreachable code, and incorrect format strings

📏gofmt Enforcement

Applies canonical Go formatting across the entire codebase. No debates about style — gofmt is the law

📊Coverage Analysis

Identifies untested code paths and generates stub test files to raise coverage toward Ma'at's 85% threshold

Usage

$ pantheon maat heal # Run Isis autonomous remediation on current project
$ pantheon maat heal --dry-run # Preview fixes without applying them
$ pantheon maat heal --lint-only # Only fix lint issues, skip vet and formatting
