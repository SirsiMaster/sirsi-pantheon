Ra

Supreme Overseer
Autonomous Hypervisor & Fleet Management
macOS ✓ Linux ✓ Windows ~

Ra is the supreme overseer — the Sirsi Hypervisor incarnated as a Pantheon deity. He orchestrates all deities across every Sirsi repository, dispatching parallel Claude agents for health checks, tests, lints, and arbitrary tasks fleet-wide. In the Ra Enterprise Edition, he manages GPU/TPU clusters, enforces compliance across routers and switches, and connects to the Sirsi Nexus platform. Ra is the crown jewel.

Usage

$ sirsi ra health # Health check across all repos
$ sirsi ra test # Run tests across all repos in parallel
$ sirsi ra lint # Run linters across all repos in parallel
$ sirsi ra task pantheon "fix the seba test failures" # Dispatch targeted task to a specific repo
$ sirsi ra broadcast "check for security vulnerabilities" # Run a prompt across all repos simultaneously
$ sirsi ra nightly # Full nightly CI: health + lint + test
$ sirsi ra status # Show orchestrator prerequisites and repo fleet

How To

Step 1 — Install prerequisites

Ra requires Python 3 and the Claude Agent SDK to dispatch parallel work.

pip3 install claude-code-sdk
Step 2 — Check status

Verify that the orchestrator, python3, and claude-code-sdk are all available.

sirsi ra status
Step 3 — Run a health check

Quick fleet-wide health check: git status, recent commits, and build verification across all repos.

sirsi ra health
Step 4 — Dispatch targeted work

Send a task to a specific repo with full tool access (Read, Grep, Bash, Edit, Write).

sirsi ra task pantheon "fix the seba test failures"
Step 5 — Enterprise integration

Ra Enterprise connects to Sirsi Nexus for fleet-wide machine management.

# Contact sirsi.ai for Ra Enterprise pricing

Frequently Asked Questions

Is Ra available now?
Yes. The Ra CLI is live as of v0.9.0. Run sirsi ra status to verify your setup, then use sirsi ra health to get started.
Will Ra be free?
The standalone Ra component (autonomous local orchestration) will be included in the free Pantheon bundle. Ra Enterprise (fleet management, Nexus integration) will be a paid tier.
Does Ra send data anywhere?
The free Ra component will be fully local with zero telemetry. Ra Enterprise will communicate with Sirsi Nexus only when explicitly configured.
What platforms will Ra support?
Ra is planned for macOS and Linux first, with partial Windows support for orchestration features that do not require platform-specific integrations.
