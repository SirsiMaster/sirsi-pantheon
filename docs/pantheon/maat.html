Ma'at is the sovereign of quality across the entire Pantheon. She runs test coverage governance, canon document verification, and CI pipeline health checks. She produces a Feather Weight score (out of 100) and enforces per-module thresholds. Her companion Scales adds YAML-based policy enforcement for organization-wide infrastructure hygiene rules.

Features

🪶Feather Weight Score

Quality score out of 100 — combines coverage, lint, canon, and pipeline metrics

📏Per-Module Thresholds

Each module has its own coverage target — enforced on push via pre-push git hook

⚖️YAML Policy Engine

Define infrastructure hygiene policies in declarative YAML — CI/CD integration ready

🔀Tiered Pre-Push Gate

Fast (10s), Standard (60s), or Deep (3-5 min) checks based on MAAT_DEPTH environment variable

📊3,667x Coverage Speedup

Diff-based coverage: only tests changed packages. 55 seconds → 15 milliseconds

Usage

$ sirsi maat # Full governance assessment
$ sirsi maat --coverage # Test coverage thresholds per module
$ sirsi maat --canon # Canon document verification
$ sirsi maat --pipeline # CI pipeline health check
$ sirsi scales enforce # Run infrastructure hygiene policies
$ sirsi scales enforce -f policy.yaml # Custom policy file

Frequently Asked Questions

Does Ma'at work on Linux or Windows?
Yes. Ma'at is pure Go and runs on all platforms where Go tests and linting tools are available.
What is the Feather Weight score?
A quality score out of 100 that combines test coverage, lint results, canon document verification, and CI pipeline health. Modules scoring below 85 cannot be included in stable releases.
Can I customize coverage thresholds?
Yes. Each module can have its own coverage target defined in your project configuration. Ma'at enforces per-module thresholds.
Does Ma'at send data anywhere?
No. All quality assessments run locally. Results are stored in local reports only. Zero telemetry.
How do I integrate Ma'at with GitHub Actions?
Add sirsi maat --fail-under 85 as a step in your CI workflow. Ma'at returns a non-zero exit code if the score is below the threshold.
