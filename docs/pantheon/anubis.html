Anubis

Judge of the Dead
Infrastructure Hygiene

Anubis is the scan and cleanup engine at the heart of Pantheon. With 58 rules across 7 domains, it discovers infrastructure waste that macOS built-in tools, CleanMyMac, and OnyX all miss — hidden caches, orphaned build artifacts, stale AI model weights, and phantom Docker volumes. Every finding includes a category, severity, and safe cleanup recommendation.

Features

🔍58 Scan Rules

Covers 7 domains: macOS caches, Docker, Dev frameworks, AI/ML, IDEs, Cloud infrastructure, and Cloud storage

🛡️Safety First

Every deletion goes to Trash with a SHA-256 decision log. 35 protected paths are hardcoded as undeletable

📊4 Scan Profiles

General, Developer, AI/ML, and Virtualization — scan what matters to you

🧠Neural Classification

Optional brain model classifies files as junk, project, config, model, data, media, archive, or essential

⚡Sub-Second Scans

Rule-based matching with early termination — most scans complete in under 2 seconds

🔌MCP Integration

scan_workspace tool available in AI IDEs via MCP server

Usage

$ pantheon weigh # Full workstation scan — discovers all waste
$ pantheon weigh --dev # Developer frameworks only (Node, Go, Rust, Python)
$ pantheon weigh --ai # AI/ML caches only (MLX, CUDA, HuggingFace, Ollama)
$ pantheon weigh --json # Machine-readable JSON output for scripting
$ pantheon judge --dry-run # Preview cleanup — shows what WOULD be deleted
$ pantheon judge --confirm # Execute cleanup — safely moves waste to Trash
