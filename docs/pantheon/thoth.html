Thoth

God of Knowledge
Persistent AI Memory & MCP
macOS ✓ Linux ✓ Windows ✓

Thoth is the knowledge keeper. He provides a three-layer persistent memory system that gives AI coding assistants instant context without re-reading thousands of source files. Layer 1 (memory.yaml) stores compressed project state. Layer 2 (journal.md) stores timestamped reasoning. Layer 3 (artifacts/) stores deep benchmarks and audits. He also runs the MCP server for integration with Claude, Cursor, and Windsurf.

How To

Step 1

Initialize in your project — Thoth creates the three-layer memory structure in your project root.

cd your-project && sirsi thoth init
Step 2

Sync memory — Update memory.yaml with current project state. Run after significant changes.

sirsi thoth sync
Step 3

Use in Claude or Cursor — Configure your IDE to use Thoth's MCP server for instant project context.

sirsi mcp
# Add to claude_desktop_config.json or .cursor/mcp.json
Step 4

Compact before context limit — When your AI session gets long, compact to preserve key context.

sirsi thoth compact
Step 5

Check memory.yaml — Review what Thoth knows about your project.

cat .thoth/memory.yaml

Frequently Asked Questions

Does Thoth work on Linux or Windows?
Yes. Thoth is pure Go with no platform-specific dependencies. It runs on macOS, Linux, and Windows.
How much context does Thoth actually save?
Measured on the sirsi-pantheon repo: without Thoth, an AI session reads 22,958 lines of source to understand the project. With Thoth, it reads 297 lines of compressed memory. Actual savings depend on your project size and AI provider pricing.
Does Thoth send my code anywhere?
No. All memory files (memory.yaml, journal.md, artifacts/) are stored locally in your project directory. Zero telemetry, zero cloud sync.
What is the difference between sync and compact?
sync updates memory.yaml with current project state. compact is used before hitting context limits to preserve key session information.
Does Thoth work with Cursor and Windsurf?
Yes. Thoth's MCP server works with any MCP-compatible IDE including Claude Code, Cursor, and Windsurf.
