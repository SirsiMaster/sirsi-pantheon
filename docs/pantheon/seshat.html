Seshat

Goddess of Writing & Measurement
Universal Knowledge Grafting
macOS ✓ Linux ✓ Windows ✓

Seshat ingests knowledge from multiple AI platforms and tools, reconciles it into a unified format, and exports to multiple targets. She is the universal bridge that ensures your knowledge travels seamlessly between Gemini, Claude, Chrome, Apple Notes, Google Workspace, NotebookLM, and Thoth.

Features

🔌5 Source Adapters

Ingests from Gemini AI Mode, Claude conversations, Chrome browsing history, Apple Notes, and Google Workspace documents

🎯3 Target Adapters

Exports reconciled knowledge to Thoth memory layers, GEMINI.md format, and Google NotebookLM sources

🌐Chrome Profile Selection

Supports multi-profile ingestion from Chrome. Select SirsiMaster or any other profile for targeted knowledge capture

🔒Secrets Filter

Automatically redacts API keys, tokens, passwords, and other sensitive data before export to any target adapter

🔄Multi-Profile Ingestion

Ingest from multiple Chrome profiles, Apple Notes accounts, and Workspace tenants in a single sweep

Usage

$ sirsi seshat ingest # Ingest knowledge from all configured source adapters
$ sirsi seshat profiles chrome # List available Chrome profiles for ingestion
$ sirsi seshat notebooklm # Export reconciled knowledge to Google NotebookLM
$ sirsi seshat open chrome --profile SirsiMaster # Open Chrome with the SirsiMaster profile for manual ingestion

How To

Step 1

Ingest knowledge sources — Pull knowledge from Gemini, Claude, Chrome, or Apple Notes.

sirsi seshat ingest --source gemini
Step 2

List profiles — See all ingested knowledge profiles and their metadata.

sirsi seshat list
Step 3

Export to NotebookLM — Send reconciled knowledge to Google NotebookLM.

sirsi seshat export --target notebooklm
Step 4

Open in browser — View the knowledge graph in an interactive browser UI.

sirsi seshat export --target browser --open

Frequently Asked Questions

Does Seshat work on Linux or Windows?
Yes. Seshat's ingestion and export pipeline is pure Go and runs on all platforms.
What knowledge sources does Seshat support?
Currently: Gemini conversation exports, Claude session data, Chrome bookmarks/history, and Apple Notes. More sources are planned.
Does Seshat send data anywhere?
Seshat exports to targets you explicitly choose (NotebookLM, browser). No automatic uploads. No telemetry. Your knowledge stays under your control.
Can Seshat merge knowledge from different AI tools?
Yes. That is Seshat's core purpose -- universal knowledge grafting. She reconciles knowledge from multiple sources into a unified format before exporting.
