SIX BIDIRECTIONAL FLOWS

Every Direction Covered

1

Gemini → NotebookLM

Extract conversations from Google Takeout and upload as NotebookLM sources for grounded research.

pipeline.py → package_notebook.py
2

NotebookLM → Gemini

Query NotebookLM and inject distilled, citation-backed answers into GEMINI.md context files.

distill_to_gemini.py
3

NotebookLM → Antigravity

Distill notebook insights into persistent Knowledge Items for cross-session AI memory.

inject_knowledge.py
4

Antigravity → NotebookLM

Export Knowledge Items and conversations as Markdown sources for NotebookLM notebooks.

sync_to_notebooklm.py
5

Gemini → Antigravity

Extract Gemini conversations and inject directly as searchable Knowledge Items.

pipeline.py --inject-ki
6

Antigravity → Gemini

Export Knowledge Items into GEMINI.md or CLAUDE.md for instant AI context loading.

sync_to_gemini.py
INSTALLATION

Three Ways to Bridge

CLI

Via the Pantheon unified binary

# Install Pantheon $ brew install sirsi-pantheon # Use Seshat subcommands $ pantheon seshat list $ pantheon seshat export --all $ pantheon seshat sync --target GEMINI.md

VS Code Extension

Activity bar + dashboard + commands

# From OpenVSX $ ext install SirsiMaster.seshat-gemini-bridge # Or from VSIX $ code --install-extension \ seshat-gemini-bridge-0.1.0.vsix

Antigravity Skill

10 Python scripts with venv isolation

# Full pipeline (extract → inject) $ python3 scripts/run.py pipeline.py \ --input antigravity \ --last 10 --inject-ki # Chrome profile selection $ python3 scripts/run.py pipeline.py \ --profile SirsiMaster
CHROME PROFILE SETUP

Multi-Profile Extraction

Seshat discovers all Chrome profiles on your system. Use --profile to select which profile to extract Gemini conversations from.

Flag Purpose Example
--list-profiles Discover all Chrome profiles pantheon seshat --list-profiles
--profile Select a specific profile --profile SirsiMaster
seshat.defaultProfile VS Code extension setting SirsiMaster (default)
