Quick start

$ sirsi scan # Find waste — caches, build artifacts, orphaned files
$ sirsi doctor # System health — RAM, disk, kernel panics
$ sirsi ghosts # Find remnants of uninstalled apps
$ sirsi quickstart # Guided first scan with recommendations

No config files. No accounts. No setup. Every command works immediately.

What it does

Ghost Detection

Finds remnants of apps you uninstalled — Launch Services phantoms, orphaned plists, Spotlight ghosts. Typically recovers 10–100 GB.

Network Security

Audits DNS, WiFi, TLS, firewalls. Auto-fixes with safety rollback — reverts within 6 seconds if anything breaks.

AI Project Memory

Persistent memory for AI coding sessions via MCP. Compresses project context to ~2% of original. Works with Claude, Cursor, Windsurf.

Hardware Profiling

Detects Apple Silicon (ANE, Metal), NVIDIA (CUDA), AMD (ROCm). Routes workloads to the fastest accelerator.

Duplicate Detection

Three-phase scan: size grouping, partial hash, full hash. Smart keep/delete recommendations across any directories.

Code Quality

Governance audits on every push. Coverage tracking, static analysis, formatting — three depth tiers from fast to deep.
