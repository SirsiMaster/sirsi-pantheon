WHAT'S NEW

Six Pillars of Progress

This release candidate brings universal knowledge grafting, modular distribution, and a completely overhauled documentation system.

01

Seshat v2.0 — Universal Knowledge Grafting

The scribe deity now bridges five source adapters and three target adapters for true cross-platform knowledge portability. Chrome profile support for authenticated sessions, NotebookLM export for AI-ready research notebooks, and an automatic secrets filter that strips API keys, tokens, and credentials before any data leaves your machine.

5 source adapters 3 target adapters Chrome profiles NotebookLM export secrets filter
02

Per-Deity Binary Builds

Install only the deities you need. Every pillar is now independently distributable via Homebrew, so you can run a single deity without pulling the full bundle.

$ brew install SirsiMaster/tools/pantheon-thoth
03

Rich CLI Help System

Run sirsi help <deity> for interactive guides, or pass the --docs flag for full terminal-styled reference pages rendered directly in your shell.

sirsi help <deity> --docs flag terminal guides
04

Sirsi Orchestrator

Parallel multi-repo automation powered by the Agent SDK. Dispatch tasks across the entire Sirsi portfolio simultaneously, with real-time streaming progress and automatic checkpoint commits.

Agent SDK parallel dispatch multi-repo
05

15 Deity Documentation Pages

Every deity now has a dedicated how-to guide, FAQ section, and platform compatibility badges. Full coverage from Anubis to Neith, with architecture diagrams and usage examples.

how-to guides FAQ platform badges
06

Getting Started Guide

A new 7-step walkthrough takes you from first install through configuration, scanning, governance, and all the way to knowledge grafting with Seshat. Zero-to-productive in under 10 minutes.

7 steps install to grafting beginner friendly
INSTALL

Get Pantheon

Install the full bundle via Homebrew in a single command.

# Tap the Sirsi repository $ brew tap SirsiMaster/tools # Install the full Pantheon bundle $ brew install sirsi-pantheon # Verify the installation $ pantheon --version

Or install individual deities: brew install SirsiMaster/tools/pantheon-thoth
