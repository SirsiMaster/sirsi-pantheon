The Pantheon

Each deity manages a domain of infrastructure hygiene

𓂀

Anubis — The Weighing

pantheon weigh · pantheon judge

Scan 7 domains and 58+ rules for infrastructure waste. Cache directories, orphaned build artifacts, stale model weights, zombie processes. Judge and safely purge what's dead.

𓂓

Ka — Ghost Hunter

pantheon ka

Hunt ghost apps — remnants of applications that were uninstalled but left behind .plist files, Launch Services registrations, and cache directories. Parallel lsregister scan in under 1s.

𓋹

Mirror — Deduplication

pantheon mirror

Find duplicate files using three-phase matching: size grouping → partial hash (first+last 4KB) → full SHA-256. Smart keep/delete recommendations. Web UI included.

𓊝

Guard — RAM Sentinel

pantheon guard

Audit memory pressure, identify zombie and orphan processes, and surgically terminate resource hogs. Process classification across 5 target types.

𓆄

Ma'at — QA Governance

pantheon maat

Automated quality assessment: coverage enforcement, canon linkage verification, lint integration. The Feather Weight score gamifies code quality.

𓇼

Scales — Policy Engine

pantheon scales

Define and enforce infrastructure hygiene policies. YAML-based rules, org-wide standards, pass/fail verdicts with actionable remediation.

𓁹

Sight — Spotlight Healer

pantheon sight

Fix ghost apps in macOS Spotlight and Launch Services. Detect phantom registrations, reindex, and restore search accuracy.

𓆣

Scarab — Fleet Scanner

pantheon scarab

Discover and audit hosts across your network. Docker container inspection, ARP scanning, port discovery. The network-scale Anubis.

𓁟

Thoth — AI Memory

pantheon mcp

Persistent knowledge system for AI coding assistants. Three-layer memory architecture achieves 98% context reduction. MCP server for Claude, Cursor, and Windsurf.
